## How to run?
### Dependencies
+ yarn

### Clone the project
```
git clone https://github.com/thorpe-innovation/chat-miauu-web.git
```
### Enter the folder
```
 cd chat-miauu-web
```
### Install the dependecies
```
yarn install
```

### Run the project
```
yarn start
```
